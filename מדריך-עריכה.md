# מדריך עריכה עצמית — האתר של גלי (galigeula.com)

מדריך קצר וברור, כדי שתוכלי (ואחרים שעוזרים לך) לשנות דברים בעצמך.
רוב הדברים נעשים מתוך מסך אחד: **מערכת הניהול (CMS)**.

---

## 1. איפה הכל מאוחסן? (התשובה לחבר שלך)

| מה | איפה |
|----|------|
| הקוד של האתר | GitHub — מאגר בשם `silverdavi/gali-cohen.com` |
| האירוח (השרת) | **GitHub Pages** — בחינם, מתפרסם אוטומטית |
| הדומיין (כתובת) | `galigeula.com` דרך **AWS Route 53** |
| התמונות והטקסטים | בתוך הקוד, בקבצי `src/content/*.yaml` |

**הרעיון:** אין שרת ואין מסד נתונים לתחזק. כל שינוי נשמר כקובץ ב‑GitHub,
ו‑GitHub מפרסם את האתר מחדש לבד תוך **דקה–שתיים**.

## 2. איך משנים תוכן? (הדרך הקלה — בלי קוד)

1. נכנסים לכתובת: **https://galigeula.com/admin**
2. לוחצים **"Sign in with Token"** → זה שולח אותך ל‑GitHub ליצור "טוקן"
   (סוג של סיסמה חד‑פעמית). מעתיקים אותו ומדביקים בחזרה. הדפדפן זוכר אותו.
   *(צריך חשבון GitHub עם הרשאת כתיבה למאגר — אצל דיויד.)*
3. בצד מופיעה רשימה של כל חלקי האתר. בוחרים, עורכים, ולוחצים **Save**.
   האתר מתעדכן לבד אחרי דקה–שתיים. צריך לרענן את הדף כדי לראות.

> כל העריכה היא **בעברית בלבד** עכשיו. אין יותר שדות באנגלית. ✅

## 3. תשובות לשאלות ששאלת

### למה הכל היה גם באנגלית?
האתר נבנה דו‑לשוני "ליתר ביטחון" לעתיד. מכיוון שאת פונה לקהל דובר עברית —
**הורדתי את האנגלית לגמרי ממסך העריכה.** עכשיו את כותבת פעם אחת, בעברית.

### איך מורידים מילים קטנות ("גלילה", "תרגול קטן")?
כבר הורדתי אותן. בעתיד, אם תרצי להסיר מילה קטנה כזו: נכנסים ב‑CMS ל
**"תוכן האתר"**, מוצאים את השדה, ו**משאירים אותו ריק** — והוא ייעלם מהאתר.

### איך משנים את התיאור בחיפוש גוגל?
ב‑CMS → **"תוכן האתר"** → **"תיאור לגוגל (SEO)"** → השדה
**"תיאור בתוצאות החיפוש בגוגל"**. זה הטקסט שמופיע מתחת לכותרת בגוגל.
**שימי לב:** גוגל מתעדכן לבד, וזה לוקח כמה ימים עד שבועות. אין כפתור "עכשיו".

### איך מורידים סעיפים שלא צריך?
- **סעיף שמבוסס על רשימה** (אירועים, מאמרים, מוצרים, פודקאסטים): פשוט
  **מוחקים את כל הפריטים** ברשימה — והסעיף כולו נעלם מהאתר אוטומטית.
- **סעיף קבוע** (כמו "תעריפים" או "שאלות נפוצות"): צריך שינוי קוד קטן.
  כבר הורדתי בשבילך את **"תעריפים"** ואת **"שאלות ותשובות"**.

### איך משנים ומשדרגים עיצוב?
עיצוב (צבעים, גופנים, מרווחים) נמצא בקוד בקובץ `src/index.css` — בראש הקובץ
יש "משתני צבע" שאפשר לשנות. זה דורש מישהו טכני. דרך ה‑CMS אפשר לשנות
**טקסטים, תמונות, מחירים וסדר** — לא את העיצוב עצמו.

## 4. מבנה האתר החדש (לפי מה שביקשת)

עמוד הבית = **התמונה שלך עם השמש + משפט עלייך**. למעלה יש כפתור **"תפריט"**
שנפתח לצד, ובו:

1. **נעים להכיר** — עלייך, המסע, האני‑מאמין (אפשר תמונה/שתיים)
2. **שירותי העסק** — טיפולים, זוגות, סדנאות, מדיטציה, מעגלי נשים
   *(לכל שירות: תיאור, תמונה, ומחיר — אופציונלי. מוסיפים/מורידים לפי תקופות.)*
3. **יומן מתעדכן** — אירועים קרובים
4. **המלצות** — בסליידר, אחת בכל פעם, עם חיצים קדימה/אחורה
5. **פודקאסט** — שני פודקאסטים לבחירה (עם טל / "ובחרת בחיים" עם לירן)
6. **בלוג ומאמרים** — מאמרים מלאים שנפתחים לקריאה בתוך האתר
7. **פרטי קשר** — וואטסאפ בולט + אייקונים של כל הרשתות
8. **חנות**

הוסר: **תעריפים**, **שאלות ותשובות**, וקטע **"טיפול / הכשרה / איפה / שפות"**.

## 5. דברים שכדאי לדעת לכל סעיף

- **שירותי העסק:** CMS → "שירותי העסק". כל שירות = שם, פרטים, תיאור, מחיר
  (אופציונלי), תמונה. להוסיף שירות = כפתור **+**; להסיר = סל המחזור.
- **אירועים:** CMS → "יומן מתעדכן". מחיקת כל האירועים מסתירה את הסעיף.
- **המלצות:** CMS → "תוכן האתר" → "המלצות".
- **פודקאסט:** CMS → "פודקאסט". פודקאסט עם **"סטטוס: בקרוב"** מוצג כ"בקרוב"
  בלי קישור. כשהוא באוויר — מוחקים את הסטטוס ומוסיפים קישור.
- **בלוג:** CMS → "בלוג ומאמרים". כותבים את הפוסט המלא בשדה **"תוכן המאמר"** —
  הוא נפתח לקריאה בתוך האתר. (שורה ריקה = פסקה חדשה.)
- **רשתות חברתיות:** CMS → "פרטי קשר ורשתות" → רשימת "רשתות חברתיות".
  **שם הרשת קובע את האייקון.** שמות נתמכים: `Instagram`, `WhatsApp`,
  `Facebook`, `YouTube`, `TikTok`, `Spotify`, `Email`. כתובת ריקה = מוסתר.
